## 使用

该模块依赖于 [phui](https://gitee.com/towardly/ph/wikis/Home?sort_id=4035190)，所以安装本模块之前需要先安装 `ph-utils`，`js` 是使用 [typescript](https://www.typescriptlang.org/) 编写。

### 安装

```
npm install ph-utils phui
```

### 引入

具体的引入请参考每一个模块的详细文档。
